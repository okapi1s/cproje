# Tufan Motor Sürücü Ödev-1

# **Araç Güvenlik Kontrol Sistemi (Start-Up Check)**

## **Bu Projede:**

## Proje amacı, elektrikli araçta motor çalıştırılmadan önce sistemdeki önemli parametreler olan:

## 1-Batarya sıcaklığı

## 2-Şarj yüzdesi

## 3-Kapı durumu

## 4-Fren pedalı durumu parametrelerini değerlendirerek sürüş güvenliği onayına karar verilen bir güvenlik algoritması oluşturulmasıdır. 

## **Kurulum**

## 1-Projeyi GitHub üzerinden bilgisayarınıza indirin:
## git clone https://github.com/kullanici-adin/repo-adin.git

## 2-Terminali açarak proje klasörüne gidin ve bu komutu yazarak kodu derleyin:
## gcc Untitled-tufan1.c -o motor_kontrol

## 3-Derleme bittikten sonra programı başlatmak için:
## Windows için 
## .\motor_kontrol.exe

## Örnek çıktı

## --- ELEKTROMOBIL GUVENLIK KONTROLU ---
## Batarya Sicakligi (C): 25
## Sarj Yuzdesi (%): 80
## Kapi Durumu (A/K): K
## Fren Pedali (1=basili, 0=degil): 1
